Getting Started
===============
Installation
------------
*ADMIRE* can be installed by cloning the repository and installing in manually::

    $ git clone https://github.com/abatten/admire
    $ cd admire
    $ pip install .

or by installing from `PyPI`_ with::

    $pip install admire
